Getting the data from the API

The API used in this data visualisation is provided by Earthquake US GOV which provide acces to the most recent earthquakes on the planet. This tutorial assumes that the reader is fluent with HTML and Javascript and knows how to read the documentation provided. GeoJSON is an open standard format designed for representing geographical features and their non-spatial attributes; as the name suggests, it is based on JSON. Within a GeoJSON file is a coordinate linking to a geographical location. Why we used it: For the Locate-a-Quake project, we used GeoJSON in conjunction with the Google Maps API to plot points on a map on the webpage The Google Maps API was used for the creation of the earthquake page and thus it played very important role for the success of this project.

Displaying in the visualisation

Framework bootstrap was used to ensure the web application was responsive. The navigation bar was provided by bootstrap and this ensures navigation of the web application is comfortable and well-defined. The fluid grid system that bootstrap implements handle viewing the application on a range of the most common display ratios, i.e. monitors and mobile phones. Bootstrap also provided a simple way to provide tooltips to some of the visualisations.
