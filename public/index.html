About the Project

This Project is part of the third year course in BSc Computing at Glasgow Caledonian University.
The scope of the project is the demonstration of Data Visualisation through the use of APIs.
As shown in the navbar above, we utilised six different APIs which show six different approaches and ways to fetch data and display this same data through the use of graphs, maps or charts.
Data visualisation is an important method of data representation which allows users to understand and manage a collection of data in an easier and more organised way.

About APIs used

This website demonstrates Data Visualisation using a list of free APIs available on the web. See below for a complete list:
1) Earthquake API
2) ISS Tracker API
3) Exchange rate API
4) Coingecko API
5) Teleport API
6) OpenWeatherMap API
