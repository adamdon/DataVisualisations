About the Project

This Project is part of the third year course in BSc Computing at Glasgow Caledonian University.
The project involves the demonstration of Data Visualisation through the use of APIs. The navbar above shows the usage of six different APIs and each one shows Visualisation through the use of charts or maps.

About APIs used

This website demonstrates Data Visualisation using a list of free APIs available on the web. See below for a complete list:
1) Earthquake API
2) ISS Tracker API
3) Exchange rate API
4) Coingecko API
5) Teleport API
6) OpenWeatherMap API
