Introduction

The following guide is aimed to beginners and should provide all the information required in order to recreate, and eventually improve, the Weather API presented in our website. You're not required to be an expert in HTML or Javascript, this guide will drive you step by step in the creation of your first API while giving you insight on the choices and steps performed.

Setting Up

In order to proceed we need to set up our workspace and for this tutorial we will make use of the free IDE Visual Studio Code. Also we want to create a new folder on the desktop, or in a directory of your choice, named "My First Weather API". Within this folder, we want to create two files: "weather.html" and "weatherdata.js".
